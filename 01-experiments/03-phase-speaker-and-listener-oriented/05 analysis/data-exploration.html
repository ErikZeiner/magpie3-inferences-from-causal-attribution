Exclusion

We exclude from further analysis participants who didn’t pass the first set of comprehension questions even on their second attempt or failed more than one question from the second set of comprehension questions.

Code 
exp1_getComprehensionPass <- function(id) {
  participant <- exp1_full |> filter(prolific_pid == id)
  comp1 <- participant |>
    filter(trial_type %in% c("comprehension-1-1", "comprehension-1-2")) |>
    group_by(trial_type) |>
    summarise(all_correct = all(correct), .groups = "drop") |>
    arrange(trial_type) |>
    pull(all_correct)

  comp2 <- participant |>
    filter(trial_type == 'comprehension-2') |>
    summarise(all_correct = all(correct), .groups = "drop") |>
    pull(all_correct)

  return(c(
    comp1_1Correct = comp1[1],
    comp1_2Correct = comp1[2],
    comp2Correct = comp2,
    compCorrect = ifelse(
      sum(c(!comp1[1], !comp1[2], !comp2), na.rm = TRUE) > 1,
      FALSE,
      TRUE
    )
  ))
}

exp2_getComprehensionPass <- function(id) {
  participant <- exp2_full |> filter(prolific_pid == id)
  comp1 <- participant |>
    filter(trial_type %in% c("comprehension-1-1", "comprehension-1-2")) |>
    group_by(trial_type) |>
    summarise(all_correct = all(correct), .groups = "drop") |>
    arrange(trial_type) |>
    pull(all_correct)

  return(c(
    comp1_1Correct = comp1[1],
    comp1_2Correct = comp1[2],
    compCorrect = ifelse(
      sum(
        c(
          !comp1[1],
          !comp1[2],
          !all(participant$correct_awareness),
          !all(participant$correct_control),
          !all(participant$correct_bg),
          !all(participant$correct_yr),
          !all(participant$correct_unlabeled)
        ),
        na.rm = TRUE
      ) >
        1,
      FALSE,
      TRUE
    )
  ))
}

exp3_getComprehensionPass <- function(id) {
  participant <- exp3_full |> filter(prolific_pid == id)

  comp1 <- participant |>
    filter(trial_type %in% c("comprehension-1-1", "comprehension-1-2")) |>
    group_by(trial_type) |>
    summarise(all_correct = all(correct), .groups = "drop") |>
    arrange(trial_type) |>
    pull(all_correct)

  comp2 <- participant |>
    filter(trial_type == 'comprehension-2') |>
    summarise(all_correct = all(correct), .groups = "drop") |>
    pull(all_correct)

  comp3 <- participant |>
    filter(trial_type == 'comprehension-3') |>
    summarise(all_correct = all(correct), .groups = "drop") |>
    pull(all_correct)

  return(c(
    comp1_1Correct = comp1[1],
    comp1_2Correct = comp1[2],
    comp2Correct = comp2,
    comp3Correct = comp3,
    compCorrect = ifelse(
      sum(
        c(
          !comp1[1],
          !comp1[2],
          !comp2,
          !comp3
        ),
        na.rm = TRUE
      ) >
        1,
      FALSE,
      TRUE
    )
  ))
}

# Generate n sample data points for a given condition from a normal distribution for a given mean and sd
exp1_get_speaker_samples <- function(
  normality,
  mechanism,
  valence,
  mean,
  sd,
  size = 30
) {
  scale <- 1:7
  probs <- dnorm(scale, mean = mean, sd = sd)
  probs <- probs / sum(probs)
  df <- data.frame(
    speaker_likelihood = sample(
      scale,
      size = size,
      replace = TRUE,
      prob = probs
    ),
    presc_normality = rep(normality, size),
    effect_valence = rep(valence, size),
    mechanism = rep(mechanism, size)
  )
  return(df)
}

    effect_valence = rep(valence, size),
    stat_normality = rep(stat_normality, size),
    mechanism = rep(mechanism, size)
  )
  return(df)
}

Hypotheses

  1. People take prescriptive norms into account (as described by the effects of abnormal inflation in conjunction and abnormal deflation in disjunction) when making causal selections, and as listeners, they infer prescriptive norms held by the speaker based on their causal selection.

[Experiment 1]

  1. People take statistical norms into account (as described by the effects of abnormal inflation in conjunction and abnormal deflation in disjunction) when making causal selections, and as listeners, they infer statistical norms held by the speaker based on their causal selection.

[Experiment 2]

  1. When both statistical and prescriptive norms are present, people’s causal ratings vary monotonically with the combined statistical-prescriptive norms, taking both into account when making causal selections such that:
  • in conjunction, likelihood ratings increase monotonically due to effect of abnormal inflation: \[[SN+PN] < [SA+PN] < [SA+PA]\] \[[SN+PN] < [SN+PA] < [SA+PA]\]
  • in disjunction, likelihood ratings decreases monotonically due to the effect of abnormal deflation: \[[SN+PN] > [SA+PN] > [SA+PA]\] \[[SN+PN] > [SN+PA] > [SA+PA]\] (SA - statistically abnormal, SN - statistically normal, PA - prescriptively abnormal, PN - prescriptively normal)

    [!TODO] When both statistical and prescriptive norms are present, listeners infer prescriptive norms from the speaker’s causal selection, and this inference is unaffected by knowledge of the statistical normality of the selected cause.

[Experiment 3]

  1. People take outcome valence into account when making causal selections, and as listeners, they infer the normality of the selected cause from it.
  • Speakers select between the abnormal and normal cause based on outcome valence, in correspondence with the ordering unpleasant > neutral > pleasant:
    • unpleasant outcomes make speakers more likely to select the abnormal cause, less likely the normal;
    • pleasant outcomes make speakers more likely to select the normal cause, less likely the abnormal;
    • neutral outcomes do not steer selection either way, falling between the two.
  • Listeners infer the (statistical/prescriptive) normality of the selected cause accordingly:
    • unpleasant outcomes make listeners more likely to infer the abnormal cause, less likely the normal;
    • pleasant outcomes make listeners more likely to infer the normal cause, less likely the abnormal;
    • neutral outcomes do not steer inference either way.

[Experiment 1, Experiment 2, and Experiment 3]

  1. [!TODO] Based on existing literature, we hypothesise that there is no effect of outcome valence (meaning that re-checking previous hypotheses stratified by outcome valence would not make a difference)
